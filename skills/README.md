# Skills

Place each video-related Codex skill in its own folder:

```text
skills/<skill-name>/
```

Each folder should be self-contained and include:

- `SKILL.md` with trigger metadata and workflow guidance.
- `agents/openai.yaml` for UI metadata.
- `scripts/` for reusable automation.
- `references/` for optional detailed guidance.

Keep generated videos, temporary work directories, caches, and media inputs out of this repository unless they are intentional fixtures.
